# Handoff Workflow

## Objetivo

Garantir continuidade entre sessões Codex e entre agentes.

## O Que Registrar

- Estado atual.
- Arquivos alterados.
- Comandos executados.
- Testes feitos.
- Testes não feitos.
- Riscos restantes.
- Próximos passos.
- Decisões tomadas.

## Exemplo de Prompt

```text
Use prompts/end-session.md para encerrar e atualizar o contexto.
```

## Arquivos

- `.ai-context/HANDOFF.md`
- `.ai-context/CHANGELOG-AI.md`
- `docs/handoff/`

## Regra

Não encerrar sessão com trabalho relevante sem atualizar handoff.
