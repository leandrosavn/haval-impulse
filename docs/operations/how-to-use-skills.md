# How To Use Skills

## Local

Skills do projeto ficam em `.agents/skills/`.

## Exemplos

```text
Use a skill haval-cluster-performance para revisar essa alteração.
```

```text
Use a skill haval-kotlin-js-bridge antes de mudar evaluateJavascript.
```

```text
Use a skill haval-docs-handoff para atualizar o handoff final.
```

## Fluxo

1. Identifique o domínio da tarefa.
2. Leia o `SKILL.md`.
3. Consulte `references/README.md`.
4. Use `examples/usage.md` para checklist prático.
5. Documente validações e riscos.
